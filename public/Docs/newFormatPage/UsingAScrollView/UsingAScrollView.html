Использование ScrollView

ScrollView – это универсальный контейнер с прокруткой, который может содержать несколько компонентов и представлений (view). Элементы с возможностью прокрутки не обязательно должны быть однородными, вы можете покручивать как по вертикали, так и по горизонтали (установив свойство Horizontal ).

Этот пример создаёт вертикальный ScrollView с картинками и текстом.

ScrollView можно настроить так, чтобы пролистывать представления (view) с помощью жестов смахивания, используя pagingEnabled prop. Свайп по горизонтали между представлениями может быть реализован и на Android, с помощью ViewPagerAndroid компонента.

ScrollView с одним элементом может использоваться, чтобы позволить пользователю масштабировать содержимое. Установите maximumZoomScale и minimumZoomScale props и ваш пользователь сможет увеличивать и уменьшать масштаб.

ScrollView лучше всего работает для представления маленького количество элементов с ограниченным размером. Все элементы и представления ScrollView отображаются, даже если их нет на экране. Если у вас большой список элементов, которые могут поместиться на экране, то лучше использовать FlatList. Итак, давайте теперь узнаем о представлениях списка.
