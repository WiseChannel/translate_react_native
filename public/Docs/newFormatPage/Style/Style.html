Style (стиль)

В React Native вы не используете специальный язык или синтаксис для определения стиля. Вы просто создаёте стиль, использую JavaScript. Все основные компоненты поддерживают prop с названием style. Имена стилей и их значения обычно совпадают с тем как CSS работает в web, за исключением имён, написанных с «верблюжьей оболочкой», например backgroundColor вместо background-color.

Style pror может быть простым старым объектом JavaScript. Это самый простой вариант и его мы используем для примеров кода. Вы также можете передать массив стилей – последний стиль в массиве имеет приоритет, поэтому вы можете использовать его для наследования стилей.
