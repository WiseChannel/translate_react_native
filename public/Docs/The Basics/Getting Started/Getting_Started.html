Начало

Эта страница поможет вам установить и сделать своё первое приложение на React Native. Если у вас уже установлен React Native, вы можете перейти к туториалу

Если вы уже занимались web разработкой, самый простой путь начать изучение React Native – это инструменты Expo, потому что они позволят вам начать проект без установки и настройки Xcode или Android Studio. Expo CLI настраивает среду разработки на вашем локальном компьютере и вы можете начать писать приложение на React Native уже через минуту. Вы можете использовать Snack, чтобы опробовать React Native прямо в браузере.

Если вы уже знакомы с нативной разработкой, вам, вероятно, захочется использовать React Native CLI. Это требует установленной Xcode или Android Studio. Если у вас уже установлена одна из этих сред разработки, вы сможете начать разрабатывать уже через минуту. Если же они не установлены, то вам понадобится около часа на установку и настройку.

Если у вас уже установлен Node 8+, вы можете использовать команду npm для установки утилиты командной строки Expo CLI:

npm install -g expo-cli

Затем следуйте за дальнейшими действиями, чтобы создать новый React Native проект с названием “AwesomeProject”:

expo init AwesomeProject

cd AwesomeProject npm start # также можете использовать: expo start

Это запустит сервер для разработки.

Запуск вашего React Native приложения

Установите Expo client приложение на ваш iOS или Android телефон и подключитесь к такой же беспроводной сети как и на компьютере. На Android используйте приложение Expo, чтобы просканировать QR код с вашего терминала для открытия проекта. На iOS, следуйте инструкциям на экране, чтобы получить ссылку.

Изменение вашего приложения

После того, как вы успешно запустили приложение, давайте изменим его. Откройте App.js в любом текстовом редакторе и отредактируйте несколько строк. Приложение должно автоматически перезагрузиться после того, как вы сохраните изменения.

Вот и всё!

Поздравляем! Вы успешно запустили и отредактировали ваше первое приложение на React Native.

Что теперь?

У Expo также есть документация которую можно почитать, если у вас появятся какие-то вопросы к инструментам разработки. Также вы можете попросить помощи на форуме Expo.

Эти инструменты помогут вам быстро начать разработку, но перед тем как приступить к созданию приложений с помощью Expo CLI, ознакомьтесь с ограничениями.

Если у вас возникли проблемы с Expo, перед тем, как создавать новую тему, посмотрите есть ли уже такая же проблема:

Запуск вашего приложения на симуляторе или виртуальном устройстве

С помощью Expo CLI легко запустить ваше React Native приложение на физическом устройстве без настройки среды разработки. Если вы хотите запустить приложение на iOS Simulator или на Android Virtual Device, пожалуйста, следуйте инструкциям для создания проектов с собственным кодом, чтобы узнать как установить Xcode или среду разработки Android.

Настроив однажды, вы сможете запустить ваше приложение на Android Virtual Device, с помощью команды npm run android, или в iOS Simulator с помощью npm run ios (только macOS)

Предостережения

Поскольку вы не создаете никакого собственного кода при использовании Expo для создания проекта, невозможно включить кастомные модули помимо API-интерфейсов React Native и компонентов, доступных в клиентском приложении Expo.

Если вы знаете, что вам понадобится подключить свой собственный нативный код, Expo всё ещё хороший способ начать работу. В таком случае, вам просто понадобится «извлечь» чтобы создать свои собственные нативные сборки. Если вы сделали извлечение, инструкции «Построение Проектов с Собственным Кодом» будут необходимы для продолжения работы.

Expo CLI настраивает ваш проект на использование самой новой версии React Native, которая поддерживается Expo client. Обычно, Expo client получает поддержку новой версии React Native через неделю после релиза стабильной версии. Вы можете проверить этот документ, чтобы узнать какие версии поддерживаются.

Если вы интегрируете React Native существующий проект, вам захочется пропустить Expo CLI и перейти непосредственно к настройке среды разработки. Выберете «Построение Проектов с Собственным Кодом» сверху, для инструкций по настройке собственной среды сборки для React Native.
