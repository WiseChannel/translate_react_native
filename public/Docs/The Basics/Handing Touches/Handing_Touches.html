Handling Touches (обработка прикосновений)

Пользователь взаимодействует с мобильными приложениями в основном с помощью прикосновений. Они могут использовать комбинацию жестов, например: нажатие кнопки, прокрутка списка или масштабирование карты. React Native предоставляет компоненты для обработки всевозможных общих жестов, а также комплексную систему реагирования на жесты, обеспечивающую ещё более расширенное распознавание жестов. Но один компонент, которые вас, скорее всего, заинтересует – то кнопка.

Отображение обычной кнопки

Кнопка обеспечивает компонент обычной кнопки, который хорошо рендерится на всех платформах. Минимальный пример отображения кнопки выглядит так:

Button onPress={() => { Alert.alert('You tapped the button!'); }} title="Press Me" />
